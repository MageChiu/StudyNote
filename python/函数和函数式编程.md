## 什么是函数式编程

指令式编程相比，函数式编程强调函数的计算比指令的执行重要。和过程化编程相比，函数式编程里，函数的计算可随时调用。


## Python中认为函数也是一个对象。

不能在函数的声明中访问属性。换句话说，在函数声明中没有“self”这样的东西让你可以进行诸如`__dict__[‘version’] = 0.1`的赋值。这是因为函数体还没有被创建，但是只有有了函数对象就可以，就可以按照上面描素的那样访问他的字典。


## 内嵌函数

在函数体内创建的另一个函数被称为内部函数或者内嵌函数


装饰器可以入函数调用一样“堆叠”起来

``` python
def counter(start_at = 0, step = 1):
    count = [start_at]
    def incr():
        count[0] += step
        return count[0]
    return incr
counter.__doc__ = """
这里counter做的唯一的事情就是接受一个初始化值来计数，
定义了一个内部函数incr，通过在incr内部使用变量count，创建了一个闭包
"""
c1 = counter(4)
c2 = counter(2)
c3_3 = counter(100,50)
print(c1())
for i in range(10):
    print(c3_3())
```

上面定义的incr函数就是一个闭包函数。如果在一个内部函数中，对外部作用域（但不是在全局作用域）的变量进行引用，那么内部函数就被认为是闭包。
闭包函数的变量可以通过使用函数的fun_closure进行追踪，在Python3中函数的该项属性更名为`__closure__`
